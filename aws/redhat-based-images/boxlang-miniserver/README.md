# BoxLang MiniServer on RedHat 8

This AMI create a runnging ColdBox Site using BoxLang with bx-compat-cfml module for you. If you do not want a ColdBox site we will show you how to remove it and have your own site. The first step is to have an AWS account. If you do not have one go to this URL to learn how to create an \[AWS account.]\( [https://aws.amazon.com/premiumsupport/knowledge-center/create-and-activate-aws-account/](https://aws.amazon.com/premiumsupport/knowledge-center/create-and-activate-aws-account/))

In AWS Marketplace you find this AMI offer as [**BoxLang MiniServer for RedHat**]() or look for **Ortus Solutions, Corp** to look for out full catalog.
